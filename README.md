# CsBinUtils
Binary utilities made by me

## Building
To build all utilities, run:
```
make
```

To build only one utility, run:
```
make <utility name>
```

## Utilities
### lc
Line counter.
Counts the lines in a text file.

### 8ball
Gives you a random answer.

### randline
Random Line.
Choses a random line from a given textfile.
